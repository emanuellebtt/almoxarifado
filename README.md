## Almoxarifado
Sistema de Almoxarifado Simples Laravel de controle de produtos e suas movimentações, desenvolvido com os alunos Senai.

### Objetivos do Sistema
Este sistema gerencia produtos de um almoxarifado, cadastrando os produtos e suas movimentações. Este sistema foi desenvolvido na discplina de Implantação de Sistemas da Firjan SENAI.
